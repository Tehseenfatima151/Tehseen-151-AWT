# Biblionex - Library Management System

A modern React.js web application for managing a university library—built for a Web Development lab project.

## Tech Stack

- **React.js** (Create React App)
- **Bootstrap 5**
- **React Router DOM**
- **Axios**
- Functional components with Hooks

## Features

- **Dashboard** – Total books, members, borrowed count, available count
- **Books** – CRUD, search, filter by category, card layout
- **Members** – CRUD, table layout
- **Borrowed Books** – Borrow/return books, view per member

## Setup

```bash
npm install
npm start
```

Runs on [http://localhost:3000](http://localhost:3000).
